list of HTML tags.

  1. !DOCTYPE html- declares the document type and version of HTML
  2. html- is the root element of the HTML document.
  3. head- contains meta-information about the document such as character encoding and viewport settings.
  4. title- sets the title of the document.
  5. body- contains the visible content of the document.
  6. h1- is a heading tag.
  7. a- is an anchor tag used for creating hyperlinks.
  8. img- embeds an image.
  9. ul- is an unordered list.
  10. li- defines list items within unordered or ordered lists.
  11. table- creates a table.
  12. tr- defines a table row.
  13. th- defines a table header cell.
  14. td- defines a table data cell.
